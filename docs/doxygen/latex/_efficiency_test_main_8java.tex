\doxysubsection{Detailed Description}
Main class for performance testing of Address\+Book operations. 

This class performs two semi-\/automated efficiency tests\+:
\begin{DoxyEnumerate}
\item Measuring the time required to load 5000 contacts from a file
\item Measuring the time to search for a specific contact in a 5000-\/contact address book
\end{DoxyEnumerate}

The tests use System.\+nano\+Time() per precise performance measurements.
