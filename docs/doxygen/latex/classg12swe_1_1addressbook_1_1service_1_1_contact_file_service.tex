\doxysubsection{Detailed Description}
Service class for handling asynchronous contact file operations. 

This class extends Java\+FX\textquotesingle{}s Service class to provide asynchronous file operations for contacts. It manages both importing and exporting contacts using a \doxylink{classg12swe_1_1addressbook_1_1service_1_1_file_service}{File\+Service}, running these potentially time-\/consuming operations in background threads to avoid blocking the UI.
