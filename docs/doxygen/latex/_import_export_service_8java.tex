\doxysubsection{Detailed Description}
Service for importing and exporting contacts in the address book application. 

This class facilitates the transfer of contact data to and from external sources, supporting various formats as needed (mainly, the v\+Card format).

Key functionalities include\+:
\begin{DoxyItemize}
\item Importing individual contacts or entire address books from external files.
\item Exporting contact data to standard formats for sharing or backup purposes.
\end{DoxyItemize}

This service builds on the file operations provided by {\ttfamily File\+Service} and integrates with the application\textquotesingle{}s controllers for seamless user interaction.
