Detailed Description

Unit tests for the AddressBook class.

This test class verifies the core functionality of the AddressBook class, including adding contacts, removing contacts, and retrieving the contact list.

The tests ensure that the AddressBook methods correctly manipulate the internal ObservableSet of Contact objects.
